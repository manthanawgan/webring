### how to add prev / next / random to your site?

the base url for the server is https://webring-vit.vercel.app. to add previous/next/random buttons to your site, use the following html snippet:

```html
<!-- replace "your-id" with the exact id from members.json -->
<a href="https://webring-vit.vercel.app/redirect?from=your-id&dir=prev">
  ← previous site</a>
<a href="https://webring-vit.vercel.app/random">
  random site</a>
<a href="https://webring-vit.vercel.app/redirect?from=your-id&dir=next">
  next site →</a>
```
